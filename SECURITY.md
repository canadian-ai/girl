# Security Policy

## Reporting Vulnerabilities

Report security vulnerabilities via the **GitHub Security tab** at:

https://github.com/anomalyco/tools/security/advisories

Do not open public issues for security vulnerabilities.

## Scope

The following are in scope:

- The `girl` CLI tool
- Library packages under `internal/`
- Machine-readable output (JSON, SARIF, GRP plans)

The following are **out of scope**:

- Private evaluation data in `evals/private/`
- Fixture files in `testdata/`

## Supported Versions

This project does not currently provide long-term support for older versions. Always use the latest release.
